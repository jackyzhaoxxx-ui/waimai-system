# 智能客服模块说明文档

## 一、功能概述

本智能客服模块是外卖系统的独立功能模块，主要特点：

1. **智能问答**：基于关键词匹配的 AI 智能回复系统
2. **聊天历史**：自动保存聊天记录，支持查看历史对话
3. **快捷问题**：预设常见问题，一键快速提问
4. **友好界面**：现代化的聊天界面，支持实时消息展示

## 二、技术架构

### 后端技术栈
- Spring Boot 2.4.5
- MyBatis-Plus 3.4.2
- MySQL 数据库
- 模拟 AI 回复（可扩展集成真实 AI 服务）

### 前端技术栈
- Vue.js
- Vant UI 组件库
- Axios HTTP 客户端

## 四、部署步骤

### 4.1 数据库配置

1. 执行 SQL 建表语句：
```bash
mysql -u root -p reggie < src/main/resources/chat_record.sql
```

或手动在数据库管理工具中执行 `chat_record.sql` 文件中的 SQL 语句。

### 4.2 启动项目

1. 确保 Maven 依赖已更新（pom.xml 已添加 Spring AI 依赖）
2. 启动 Spring Boot 应用
3. 访问 http://localhost:8080/front/index.html

### 4.3 使用流程

1. 登录用户账号
2. 进入个人中心（点击用户头像）
3. 点击"联系客服"按钮
4. 进入聊天界面开始对话

### 5.2 API 接口

#### 5.2.1 发送消息
- **接口**: POST `/chat/send`
- **请求头**: `userId: 用户 ID`
- **请求体**:
```json
{
  "question": "用户问题",
  "sessionId": "会话 ID"
}
```
- **响应**:
```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "answer": "AI 回复内容",
    "sessionId": "会话 ID"
  }
}
```

#### 5.2.2 获取聊天记录
- **接口**: GET `/chat/history?sessionId=会话 ID`
- **响应**: 聊天记录列表

### 5.3 前端功能

1. **实时聊天**: 输入框输入问题，点击发送获取 AI 回复
2. **快捷问题**: 点击预设问题快速提问
3. **历史记录**: 自动加载当前会话的历史记录
4. **消息展示**: 气泡式消息展示，区分用户和 AI 消息
5. **加载动画**: AI 思考时的打字动画效果

## 六、扩展升级（可选）

### 6.1 集成真实 AI 服务

如需集成真实的 AI 服务（如 OpenAI GPT、通义千问等），请：

1. 移除 pom.xml 中的 Spring AI 依赖注释
2. 在 application.yml 中配置真实的 API Key
3. 修改 ChatServiceImpl 中的 chat 方法，使用真实的 AI 客户端

示例代码（ChatServiceImpl.java）:
```java
@Autowired
private ChatClient chatClient; // Spring AI 的 ChatClient

@Override
public ChatResponse chat(Long userId, String sessionId, String question) {
    String answer = chatClient.prompt()
            .system(systemPrompt)
            .user(question)
            .call()
            .content();
    // ... 保存记录等后续操作
}
```

### 6.2 自定义回复

可以通过以下方式优化回复质量：

1. **扩展 FAQ_MAP**: 在 ChatServiceImpl 中添加更多关键词和回复
2. **调整 systemPrompt**: 在 application.yml 中修改系统提示词
3. **添加上下文理解**: 实现多轮对话，记住之前的聊天内容

## 八、常见问题

### Q1: 为什么使用模拟 AI 而不是真实 AI？
A: Spring Boot 2.4.5 版本较老，Spring AI 需要更高版本。模拟 AI 可以演示功能，同时保留扩展性。实际项目中可以轻松替换为真实 AI 服务。

### Q2: 如何测试？
A: 
1. 启动项目后访问前端页面
2. 登录任意用户账号
3. 进入个人中心点击"联系客服"
4. 尝试提问："配送多久"、"有什么优惠"、"推荐菜品"等

### Q3: 聊天记录保存多久？
A: 目前永久保存，可以通过添加定时任务定期清理过期记录。
